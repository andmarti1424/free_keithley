# free_keithley
Plotting / measurement software for Keithley 2015 THD multimeter

## Utilities
 - THD measurement: THD/THDN and distortion of each harmonic
 - Vdc logger
 - THD vs Freq.
 - Freq. response plot
 - THD vs Power: PLANNED

## Requirements
 - `python 3.10+`
 - `matplotlib`
 - `python-pyserial`
 - `tkinter (tk)`

## Usage
```python thd.py```
or
```python logger.py```
